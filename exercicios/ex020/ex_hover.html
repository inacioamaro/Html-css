No CSS, o Hover CSS faz parte do conjunto de palavras-chave utilizadas pelas pseudo-classes da linguagem de estilos e serve para adicionar características a um elemento quando a pessoa usuária posiciona o mouse sobre ele

        
div > p {
            display: none;
        }

Indica que o elemnto pretencente a div (nesse caso o "p") e pretencente a div e nesse caso do display none o elemento sera retirado da tela (ficara escondido)

        div:hover > p{
            display: block;
            color: white;
            background-color: red;
            width: 300px;
        }

Indica que ao passa o molse por cima do elemento ira mostra o elemento com a seguintes características a cima citada especifica para o elemento pertencente ao div
                
        div:hover{
            color: yellow;
            background-color: black;
            width: 300px;
        }
