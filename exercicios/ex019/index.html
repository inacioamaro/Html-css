Identificando os elementos iguais com "id"

Podemos identificar os elementos iguais e diferenciar los para acrescentar formatação e ele de destacar dos demais / Nesse exemplo temos 2 H1 onde se quiser colocar apenas o 1º centralizado posso usar um id="nome que você escolher".
Por regra não devemos colocar o mesmo "id" em vários elementos do nosso site. (fusiona, mas não é profissional segundo o W3C)

  • tudo html "id" em css representa como cerrilha "#"
  • tudo html "class" em css representa como ponto"."

Classificando os elementos iguais com "class"

Quando tenho dois ou mais elementos que queira aplicar a mesma formatação diferenciada não vou usa a "id" uso o "class"

Misturando

Podemos usar mais de uma classe no mesmo elemento desde que a formatação não entre em conflito Ex; na primeira class color "blue" segunda class color "black" isso daria um conflito e iria escolher o último comando e podemos colocar um id e class no mesmo elemento respeitando a mesma regra citada anteriormente
