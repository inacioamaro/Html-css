1.1 Codigo-fonte / Pre formatado

A tag "code" é usada para organizar o espaçamento
Exemplo sem o cod: O comando docuent.getElementById('teste') é escrito em JavaScript.

Exemplo com o code: O comando docuent.getElementById('teste') é escrito em JavaScript. com fonte mono espaçada

Mas devemos ter cuidado com esse comando pos nem sempre devemos usar Exemplo comando pyton com o code :

num=int(imput('Diite um numero')) if num % 2 == 0: print(f'O numerio {num} é par') else: print(f'O numero {num} é impar') print('Fim do programa')

O code nãp respeita a identação para que isso ocora pode mesclar a tag pre com o code

        
num=int(imput('Diite um numero'))
if num % 2 == 0:
    print(f'O numerio {num} é par')
else:
    print(f'O numero {num} é impar')
print('Fim do programa')
        
        
Vemos agora que a identação foi respeitada
