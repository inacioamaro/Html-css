Estilos CSS internos

No estilo de CSS internos ele e aplicado no "head" acima ou abaixo do title onde geramos uma tag style e informamos qual e a tag que iremos formata Ex:

    Style
    body{
        background-color: lightskyblue; font-family: Arial, Helvetica, sans-serif; font-size: 20px;
    }
    h1{
        color: mediumblue; background-color: lightblue;
    }
    h2{
        color: darkred;
    }
    Style

entre outros Estilos que podemos criar ...
