1. Principais formatações

1.1. Negrito / Destaques

Podemos usar duas formas de colocar uma palavra em negrito semântica ou não semântica

Na forma não semântica usamos a tag <b>

Na forma semântica usamos a tag <strong>
Sendo que na tag u a expressão tem como finalidade de forma apenas e na tag strong dá sentido de destaque que é semântica

1.2. Itálico / Enfase

Podemos usar a tag i para colocar em Itálico

Na forma semântica devemos usar a tag "em" para gerar Enfase com semântica

Devida a nova forma que utilizar o html5 é solicitado que use as tags de semântica já que as tags não semânticas são tags antigas e devem entrar em desuso em breve

1.3. Texto marcado

podemos criar um texto mancado usando a tag mark
Lembrando que com a tag mark não e possivel mudar a cor de sua marcação pois ela e não é forma e sim um significado "semantico"

Caso deseje escolher a cor do meu marcado tenho que usar CSS3.que me permite formta atraves do paramentro style "background-color" e seleciona a cor desejada dentro de mark

Caso se o documento tenha varias marcações e queira que todar herdem a mesma configuração não se pode fazer a configuração em apenas um paragrafo deve criar um tag style dentro dele criar um seletor para mark no cabeçario "head" com a cor desejada
Com isso sempre que usar a tag MARK. ela herdara a formatação desejada

1.4. TEXTO grande e texto pequeno

Para cria um texto grando ou pequeno usamos as tags "big" para almentar o tamanha das letras e "small" para diminuir o tamanha das letras
Hoje em dia usamos a formatação em cabeçãrio que veremos em um materiol a frente

1.5. Texto deletado

Para informa que aquele texto não esta mais em uso pode ser usado a tag "del" o txto pode ser lida mas não deve ser levado em consideração!

1.6. Texto incerido

Para informar que um texto foi incerido no documento podemos usar a tag "u" ou ins indica que o texto foi inserido para dar uma enfase e indicar que foi adicionado depois
Lembrando que a teg u esta defasada (não semantica)

1.7. Texto sobrescrito

O texto sobrescrito é o texto que usa por exemplo (x20+3) usa-se a tag sup

1.8. Texto subescrito

Caso queira subescrever algo exemplo(H2O)usa-se a tag sub
